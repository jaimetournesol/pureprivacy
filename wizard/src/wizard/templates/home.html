2
Open Element once, then start Orbot
Open Element on your phone briefly so Orbot can see it in the app list, then close it.
Open Orbot → tap Start → allow the VPN prompt → wait for Connected to the Tor network.
(Optional: in Orbot, switch to "Tor-Enabled Apps" mode and tick only Element — saves battery.)

Add this server in Element
